CSS Selector Cheatsheet

Selector Table Guide

Tag : Syntax : What It Does :
div N/A Content DIVision tag.Selects all div tags
div , span ',' comma Selects all of the specified elements. In this case: Selects all div and span elements.
div span div [space] span Selects all the elements, in this case: span elements, in a div.
#content '#' hashtag Makes use of the HTML ID selector used to target a single element with a unique id. In this case: An HTML ID of Content.
.box '.' period Selects all elements with that class attribute to apply unique CSS declarations.Also emphasizes the importance of semantic CSS aka descriptive code names.
div > p > Greater Than Selects all elements, in this case paragraphs, where the parent is a div element.
div + p '+' Plus Sign Selects the first element placed immediately after the div element.In this case: Selects the first paragraph element.
p ~ ul '~' Tilde Selects every element, in this case an Unordered List, preceded by a paragraph element

Pseudo-class Selector Table

Tag : Syntax : What It Does :
a:link 'a' stands for anchor. The basic syntax is selector : pseudoclass . Defines how an unvisited link should be styled.
a:visited N / A Defines how a visited link will be styled.
a:hover a:hover must come after both a:link and a:visited in order for the CSS styling to work. Styles how a link will look on mouse hover.
div: hover div:hover { background-color: blue; } Pseudo-classes can be combined with other elements. In this case: when the div is hovered over it will change its color to blue.
p : first-child p:first-child { color: blue; } In this case this would style the first paragraph element in the color blue. If you had two paragraph elements the first one would be blue and the second would be whatever other standard you set in your CSS
